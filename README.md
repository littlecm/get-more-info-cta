# Get More Information CTA

This project is a Next.js app that provides a "Get More Information" CTA for vehicle detail pages on dealership websites. The form collects customer information and sends it to the dealership via ADF/XML.

## Getting Started

1. Clone the repository.
2. Install dependencies with `npm install`.
3. Set up environment variables in `.env.local`.
4. Run the development server with `npm run dev`.

## Deployment

This app can be deployed to Vercel for easy scaling and management.

## Environment Variables

- `AUTH0_CLIENT_ID`
- `AUTH0_CLIENT_SECRET`
- `AUTH0_DOMAIN`
- `NEXTAUTH_SECRET`
- `SUPABASE_URL`
- `SUPABASE_KEY`
- `DATABASE_URL`
